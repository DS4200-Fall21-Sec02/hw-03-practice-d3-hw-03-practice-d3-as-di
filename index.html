The data for the scatter plot was iris data. We chose this because it was an interesting dataset that we discovered on Kaggle. The iris data is suitable for a scatterplot because both variables (Sepal Width and Sepal Length) are quantitative. The data for the grouped bar chart was housing data. This was also chosen while browsing through Kaggle. This dataset seemed unique, so that is why it was chosen. A grouped bar chart is suitable for this because it has two categorical variables and one quantitative value. One of the categorical data would be used for the x-axis and the other categorical data would be indicated by color. The quantitative data would be indicated on the y-axis.
